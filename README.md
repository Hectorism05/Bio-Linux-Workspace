# Bio-Linux-Workspace 🧪💻

### 🛠️ Especificaciones del Sistema
* **SO:** Linux Mint 21.3 x86_64 Cinnamon
* **CPU:** AMD Ryzen 7 5700U with Radeon G 
* **RAM:** 12 Gb

---
## 🚀 Objetivo del Proyecto
Optimizar el flujo de trabajo para estudiantes de Ingeniería y entusiastas de Linux, priorizando la estabilidad y el ahorro de recursos.

## 📂 Contenido
1. **Sistema Base:** Post-instalación.
2. **Energía:** Optimización de batería (TLP/Powertop).
3. **Interfaz:** Personalización estética y funcional.
4. **Programas:** Alternativas Open Source.
